Sistema de Controle Financeiro para Jovens e Adolescentes

Trabalho de Conclusão de Curso - TCC

Sobre o Projeto

O sistema foi desenvolvido para auxiliar jovens e adolescentes na organização financeira, permitindo o controle de receitas, despesas, metas financeiras, contas bancárias e conteúdos de educação financeira.

Funcionalidades

  • Cadastro de Usuários
  • Login
  • Controle de Contas Bancárias
  • Transações Financeiras
  • Metas Financeiras
  • Grupos Financeiros
  • Educação Financeira

Tecnologias Utilizadas

  • PHP
  • MySQL
  • HTML5
  • CSS3
  • JavaScript
  • Bootstrap
